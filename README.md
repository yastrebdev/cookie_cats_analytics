# Cookie Cats — A/B Test Analysis

Анализ пользовательского поведения в мобильной игре Cookie Cats на основе проведённого A/B теста.

---

## Описание проекта

Cookie Cats — популярная мобильная головоломка. В игре есть механика «ворот» (gates): на определённом уровне игрок обязан сделать паузу или заплатить, чтобы продолжить.

В данном эксперименте тестировались два варианта размещения ворот:
- **gate_30** — ворота на 30-м уровне (контрольная группа)
- **gate_40** — ворота на 40-м уровне (экспериментальная группа)

**Цель:** определить, влияет ли позиция ворот на удержание пользователей и игровую активность.

## Описание данных

| Поле | Тип | Описание |
|------|-----|----------|
| `userid` | int | Уникальный идентификатор пользователя |
| `version` | str | Группа A/B теста: `gate_30` или `gate_40` |
| `sum_gamerounds` | int | Суммарное количество сыгранных раундов |
| `retention_1` | bool | Вернулся ли пользователь на 1-й день после установки |
| `retention_7` | bool | Вернулся ли пользователь на 7-й день после установки |

**Объём выборки:** 90 189 пользователей
- gate_30: 44 700
- gate_40: 45 489

---

## Методология

### Описательный анализ
- Сравнение распределений `sum_gamerounds` между группами (среднее, медиана, мода)
- Расчёт долей retention по каждой группе
- Визуализация: гистограммы, boxplot, bar chart

### Статистические тесты

| Метрика | Тест | Обоснование |
|---------|------|-------------|
| `retention_1`, `retention_7` | Z-test для долей | Бинарные переменные, большие выборки → CLT выполнен |
| `sum_gamerounds` | Критерий Манна-Уитни | Сильно скошенное распределение, нормальность не выполнена |

**Уровень значимости:** α = 0.05

---

## Ключевые результаты

| Метрика | gate_30 | gate_40 | Значимость |
|---------|---------|---------|------------|
| Retention Day 1 | ~44.8% | ~44.2% | ❌ Нет (p > 0.05) |
| Retention Day 7 | ~19.0% | ~18.2% | ✅ Есть (p < 0.05) |
| Median game rounds | 17 | 16 | — |

### Основные находки:
- Позиция ворот **не влияет** на однодневное удержание
- Gate_30 **значимо лучше** удерживает пользователей на 7-й день (+0.8 п.п.)
- Игровая активность (количество раундов) практически не отличается между группами
- В обеих группах среднее ~в 3x больше медианы — есть «хвост» из очень активных игроков

---

## Рекомендация

> **Оставить gate_30 (контрольную группу)**

Семидневное удержание — ключевой показатель долгосрочной вовлечённости. Пользователь, вернувшийся через неделю, с высокой вероятностью становится регулярным игроком. Перенос ворот на уровень 40 ухудшает этот показатель без компенсирующего прироста в активности.

## Ограничения

- Отсутствуют данные о монетизации — неизвестно, как версия влияет на платящих пользователей
